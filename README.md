# Smart Classroom Timetable Scheduler

An intelligent system designed to automatically generate and manage classroom timetables efficiently.
The system reduces scheduling conflicts, optimizes resource utilization, and simplifies timetable management for educational institutions.

---

## 📌 Overview

Creating academic timetables manually is complex and time-consuming.
This project automates the scheduling process using algorithmic logic to assign:

* Faculty
* Subjects
* Classrooms
* Time slots

while avoiding conflicts and maximizing efficiency.

---

## 🚀 Features

* Automatic timetable generation
* Conflict-free scheduling
* Faculty and classroom allocation
* User-friendly interface
* Efficient timetable management
* Easily extendable system

---

## 🧠 System Architecture

The system follows a modular structure:

Frontend
Handles user interaction and timetable display.

Backend
Processes scheduling logic and handles requests.

Database
Stores information about teachers, classrooms, subjects, and schedules.

## 🔧 Installation

Clone the repository

git clone https://github.com/Tanya426/smartclassroom_timescheduler.git

Navigate to the project directory

cd Smart-Classroom-Timetable-Scheduler

Install dependencies

pip install -r requirements.txt

Run the application

python app.py
